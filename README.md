# Alta Daily

Alta Daily es un prototipo web de estilista personal con IA: digitaliza prendas, recomienda looks diarios segun clima, agenda y estilo, y guarda combinaciones en un calendario semanal.

## Abrir la app

Abre `index.html` en el navegador. No necesita instalar dependencias.

## Funciones incluidas

- Armario digital con prendas de ejemplo.
- Carga de fotos o imagenes de ropa desde tu computador.
- Recomendaciones automaticas por clima, agenda y estilo.
- Priorizacion de prendas poco usadas para aumentar la rotacion del armario.
- Guardado de looks en un calendario semanal.
- Contadores de uso, prendas poco usadas y looks guardados.

## Backend

La carpeta `backend/` contiene el primer diseno tecnico del backend:

- `backend/schema.sql`: modelo relacional PostgreSQL.
- `backend/API.md`: contrato inicial de endpoints.
- `backend/README.md`: stack recomendado y orden de implementacion.
